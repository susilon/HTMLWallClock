# HTMLWallClock
HTML Wall Clock for your websites

demo : https://piapiastudio.web.id/htmlwallclock/

example parameter in query string :
Size
clock-size=500px

Base Color
circle-fill=blue

Base Line Color
circle-stroke=yellow

Second Hand Color
second-color=red

Second Hand Width
second-width=3px

Minute Hand Color
minute-color=green

Minute Hand Width
minute-width=5px

Hour Hand Color
hour-color=yellow

Hour Hand Width
hour-width=7px

Hour Line Color
line-color=yellow

Hour Line Width
line-width=3px

example : https://piapiastudio.web.id/htmlwallclock/?clock-size=600&second-color=yellow
